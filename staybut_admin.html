데이터 가져오기
부킹닷컴 .xls 또는 CSV 파일을 업로드하면 자동으로 분석해서 어드민에 추가합니다
다운로드 위치: 부킹닷컴 익스트라넷 → 예약 → 우측 상단 "다운로드" 버튼 (.xls)
자동 입력: 수수료율·결제대행수수료(매출의 2.5%)·취소일·도착예정시간·국적·기기 등 26개 컬럼 자동 매핑
참고: 취소된 예약(cancelled_by_guest)은 자동으로 cancelled 상태로 저장되며 청소 작업이 생성되지 않습니다.
다운로드 위치: 에어비앤비 호스트 → 수익 → 거래 내역 → CSV 다운로드
자동 매핑: 예약코드 · 시작일/종료일 · 게스트명 · 리스팅의 (호실) · 금액(매출=호스팅 총수입, 실수령, 서비스 수수료)
참고: 이미 등록된 예약번호(메일 자동등록 포함)는 자동으로 제외됩니다. 객실은 리스팅명의 (202호)에서 자동 인식됩니다.
다운로드 위치: 아고다 YCS(익스트라넷) → 예약 → 리포트 다운로드 요청 → 계정 메일(seongsu@staybut.net)로 도착한 CSV 첨부파일
자동 매핑: 예약번호 · 체크인/체크아웃 · 게스트명 · 국적 · 객실유형(호실 자동인식) · 매출(ReferenceSellInclusive) · 수수료(Commission) · 결제수수료(부가세) · 도착예정시간
참고: 이미 등록된 예약번호는 자동 제외되고, 취소/노쇼 예약은 cancelled로 저장됩니다. 객실은 룸타입으로 자동 인식됩니다(101·102·201·202·301). 못 알아본 건만 미리보기에서 직접 선택해주세요.
지원 컬럼: 체크인 · 체크아웃 · 객실(또는 listing) · 매출(또는 gross/price/amount) · 수수료(또는 fee/commission) · 예약번호(또는 code/confirmation)
인코딩: UTF-8 권장. 깨지면 엑셀에서 "다른 이름으로 저장 → CSV UTF-8"로 다시 저장해주세요.
